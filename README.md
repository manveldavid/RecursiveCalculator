# RecursiveCalculator

Try it: https://t.me/RecursiveCalcBot

It helps you solve mathematical expressions.
Like this: `1368-(0,0267+0,38/0,14+0,05/0,3+0,12/0,15+0,02/0,09)/2,94*(1368-280)` or `1+cos(pi()/3)^2/sin(2/pi())^2`

Operator list:
- `!()` - factorial
- `abs()` - takes value without sign
- `ctg()`, `tg()`, `cos()`, `sin()`, `tgh()`, `cosh()`, `sinh()` - radian unit argument
- `arctg()`, `arccos()`, `arcsin()` - radian unit returns
- `sqrt(;)` (or `√()`, `∛()`, `∜()`) - root (right part is root degree)
- `pow()` - power
- `%()` - division remainder
- `exp()` (or `e()`) - exponent
- `pi()` - pi

Digit list:
- `+` - addition
- `-` - substraction
- `/` - division
- `*` - multiplication
